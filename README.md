# KickFlip

KickFlip is an app designed to help skaters track their progress and organize clips, data, and future ideas all in one place.

## Features
- Personal vault for each user to store tricks and related videos  
- Trick details popup showing title, date, location, and video of the landed trick  
- Progress tracking over time with algorithms to plan future improvements  
- Weather forecasts to identify skateable areas based on rain conditions  

## Tech Stack
- Android Studio  
- Java  
- Firebase  
- XML  

## Current Status
- Core functionality implemented: user authentication, vault, trick uploads, and navigation  
- Additional pages and UX improvements are planned
